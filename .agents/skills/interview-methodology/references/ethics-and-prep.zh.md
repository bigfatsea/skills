# 伦理框架与访前准备

SKILL.zh.md 模式一的详细参考文件。生成访前研究报告、访谈大纲或伦理核查清单时读取此文件。

---

## A. 伦理与法律框架

### A.1 六步知情同意流程

| 步骤 | 动作 | 文档 | 时间节点 |
|---|---|---|---|
| 1 | 准备项目说明（目的、范围、用途、风险） | 项目说明书 | 首次接触前 |
| 2 | 准备录制同意书（含退出权、化名权、限制访问权） | 录制同意书 | 首次接触前 |
| 3 | 准备授权发布书（版权归属、公开范围） | 授权发布书 | 首次接触前 |
| 4 | 预访谈时逐项说明，给受访者阅读时间 | 口头 + 文件 | 预访谈时 |
| 5 | 录制开始前口头再次确认同意，录音记录 | 录音中 | 每次录制开始 |
| 6 | 访谈结束后签署授权发布书 | 授权发布书签字 | 访谈结束后 |

**为何步骤 5 先于步骤 6：** 许多口述史从业者倾向于在访谈*结束后*再签授权发布书，以便在签署时处理访谈中出现的敏感内容。但录制同意书必须在录制*开始前*取得。（OAH Best Practices）

### A.2 受访者三项权利 — 录制前必须告知

1. **退出权** — 受访者有权在任何时刻拒绝讨论任何话题
2. **化名权** — 受访者有权选择以化名出现
3. **限制访问权** — 受访者有权要求对访谈部分或全部内容限制访问

这些不是可选的形式，是知情同意的核心内容。未告知即无效同意。

### A.3 敏感/受监控社群的特殊处理

当受访者来自政治、法律或社会处境脆弱的社群时：只收集最低限度必要背景信息，不记录可能造成风险的识别性信息。（OAH Best Practices）

### A.4 受访者主观性 — 特性而非缺陷

口述史告诉我们的，与其说是事件本身，不如说是事件的*意义*。受访者的主观性正是口述史源头独特而珍贵的地方。（Alessandro Portelli）

不同场次对同一事件给出不同版本，不是不诚实——这是*叙事重构*。带时间戳记录矛盾，不要将其统一成单一权威版本。追求受访者的真实，而非客观事实。

### A.5 法律效力说明

口述访谈是具有法律严肃性的文件。转写稿和编辑稿版权通常归属采访员或项目方（取决于合同）；受访者的口述事实陈述本身通常不独立享有版权。涉及第三方的内容（尤其是负面描述）可能产生诽谤或隐私风险——出版前须评估。

### A.6 伦理自检清单（每次录制前执行）

- [ ] 受访者是否完全理解项目目的？
- [ ] 受访者是否知道录音会被如何使用？
- [ ] 受访者是否知道可以随时退出、使用化名、限制访问？
- [ ] 我是否在追求受访者的真实，而非「客观的事实真相」？
- [ ] 我是否警惕对受访者强加自己的解读？
- [ ] 我是否准备好给予受访者应有的尊重（不打断、不评判）？
- [ ] 我是否有管理自身情绪反应的机制（采访员笔记）？

## B. 访前准备

### B.1 采访员培训清单

首次上岗前，采访员应具备以下基础：

- [ ] 知情同意流程（上述 A.1–A.3 步骤）
- [ ] 录音设备操作 + 备份流程
- [ ] 主动倾听技巧
- [ ] 追问技巧
- [ ] 敏感话题应对（创伤、宗教、政治、死亡）
- [ ] 转写/整理基础

### B.2 两次预访谈流程

**第一次 — 项目说明：**
- 自我介绍 + 项目背景
- 解释录制流程、时长、用途
- 说明受访者权利（退出/化名/限制）
- 取得初步（口头）同意
- 询问受访者希望贡献或实现的目标

**第二次 — 预演 + 期望确认：**
- 讨论大纲*主题词*（不是具体问题）
- 确认「必须谈到」和「不能谈」的内容
- 询问首选称谓
- 了解受访者的口音/方言/特殊说话习惯（为转写校准）
- 确认时间、地点、设备

**两次预访谈的目的：** 「这些会面对于建立采访员与叙述者之间的信任关系至关重要。」（OAH Best Practices）——不要把首次接触当作信息获取，要当作信任建立。

### B.3 访前研究维度表格（访谈前填写）

| 维度 | 查找内容 | 建议信源 |
|---|---|---|
| **背景** | 年龄、出生地、家庭结构、教育、职业 | 公开资料、社媒 |
| **关系图** | 核心家人、朋友、同事、对手、导师 | 受访者自述、公开资料 |
| **时间线** | 人生大事年表（时间顺序） | 自传/履历/新闻存档 |
| **关键事件** | 已知转折点（婚恋、职业变化、创伤、成就） | 新闻报道、社媒 |
| **历史语境** | 受访者经历的年代与社会政治背景 | 史料、新闻 |
| **争议点** | 涉及受访者的已知争议或事件 | 公开信息（谨慎处理） |
| **软信息** | 口头禅、爱好、独特习惯 | 社媒、亲友圈 |
| **法律/伦理预警** | 是否涉及第三方、未公开信息、未审结案件 | 内部审查 |

### B.4 大纲构建方法

**原则：** 从主题词出发，而不是从预设问题出发。

- 主题词 = 一个经历域（如「童年」「职业转折」「与某人的关系」）
- 问题从主题衍生；问题本身不是主题
- 使用开放性问题（「什么」「怎样」）——不用封闭式或引导性问题
- 每个问题标注目的：`[目的：暖身/场景建立/关键事件/反思]`

**结构规则：**
- 不超过 2 页 A4
- 灵活时间块——不把 90 分钟排满
- 结尾预留 15 分钟供受访者补充未涉及内容
- 准备 5 个应急问题（用于冷场或不愿深聊时）
- 访谈时使用纸质大纲（非电子）——可在旁边手写，维持眼神接触

**每个主题词下的四类问题：**

| 类型 | 示例 | 目的 |
|---|---|---|
| 场景开场 | 「你对那个地方最早的记忆是什么？」 | 暖身，建立感官基础 |
| 关系 | 「那时候你身边都是什么样的人？」 | 填充关系图 |
| 关键事件 | 「发生了什么让事情走向改变？」 | 寻找转折点 |
| 反思 | 「现在回看那段时期，你怎么看？」 | 浮现意义与主题 |

### B.5 大纲示例（人生经历类 — 简化版）

```
主题 1：童年  [15 分钟]
- 你最早的记忆是什么？  [目的：暖身]
- 你在哪里长大？周围是什么样的世界？  [目的：场景建立]
- 童年时你的父母/抚养者是什么形象？  [目的：关系图]
- 童年的哪个时刻对你影响最深？  [目的：关键事件]

主题 2：教育/职业转折  [20 分钟]
- 你是怎么走向这个领域/工作的？  [目的：决策点]
- 转折在什么时候？为什么？  [目的：关键事件]
- 那时候最大的障碍是什么？  [目的：冲突/挑战]
- 从那段经历里你学到了什么？  [目的：反思/主题]

主题 3：核心关系/事件  [20 分钟]
- [根据访前研究确认的核心关系展开]
- ...

主题 4：当下与回望  [10 分钟]
- 从今天的位置看，你怎么看那段时期？  [目的：回顾意义]
- 如果和年轻的自己说一句话，你会说什么？  [目的：内在主题]

主题 5：补充  [15 分钟]
- 还有什么我们没谈到但你认为重要的吗？  [目的：兜底]

应急问题（冷场/不愿深聊时使用）：
1. 说说那段时间一个典型的日子是什么样的。
2. 那个章节里对你最重要的人是谁？
3. 有没有一个你这些年反复想起的时刻？
4. 那时候你想要什么但没有？
5. 什么是外人意想不到的你在那段时期的样子？
```
