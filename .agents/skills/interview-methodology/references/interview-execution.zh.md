## C. 访中执行

### C.1 录制前检查清单（提前 30 分钟）

- [ ] 检查设备电量、存储容量、清空旧录音
- [ ] 测试录音电平（正常说话距离，峰值 -6dB 至 -12dB）
- [ ] 选择安静位置；关闭门窗/空调；静音手机
- [ ] 准备纸质大纲 + 笔（不用电子——纸质可在旁边手写，维持眼神接触）
- [ ] 准备水和纸巾（情感触发应急预案）
- [ ] 准备同意书文件

### C.2 录制 Lead-in（对着设备说）

在正式内容开始前，录音设备上录制以下内容：

> 「今天是 [日期，日/月/年]。访谈受访者 [姓名或化名]，采访员 [姓名]。地点：[通用描述——如「受访者家中」，不写具体住址]。主题：[本次访谈主题]。预计时长：[X] 分钟。开始时间：[时:分]。」

这为存档和后续转写提供了永久性的上下文记录。（OAH Best Practices 要求）

### C.3 现场执行规则

| 规则 | 反例（不要做） | 正例（应该做） |
|---|---|---|
| 用「什么」而非「为什么」 | 「你为什么离开那家公司？」 | 「你离开那家公司的时候，发生了什么？」 |
| 用「怎样」而非「是不是」 | 「你是不是很伤心？」 | 「那个时候你心里在想什么？」 |
| 让沉默存在 | 立刻填充沉默 | 等 5–10 秒再介入 |
| 不打断 | 「等等，我先把问题问完」 | 让他说完；再说「我想回到你刚才说的 X」 |
| 不投射 | 「我也有类似的经历……」 | 保持聚焦受访者 |
| 不灌输 | 「所以你是说这是不公平的？」 | 「我想确认一下，我听到的是 X，是这样吗？」 |
| 应对回避 | 立刻放弃话题 | 「你刚才提到了 X，我对那个挺好奇的」（试一次；若仍回避，标记并继续） |

### C.3.A 主动倾听 — 基础能力

有效访谈的本质是**八成在听，两成在问**。采访员的工作不是审讯，而是创造条件让叙述者讲出自己的故事。

主动倾听有三个构成要素（Rogers & Farson）：

| 要素 | 含义 | 实践方法 |
|---|---|---|
| **理解（Comprehension）** | 理解被传达的内容，而非只是字面的词语 | 时刻问自己：「叙述者真正在告诉我什么？」 |
| **留存（Retaining）** | 在整个访谈过程中记住对方说的话，不因记录而打断 | 用纸质大纲旁边手写；不要中断眼神接触 |
| **回应（Responding）** | 传达「我听到了，我理解了」的信号——三种回应方式见下 | |

**三种主动回应方式：**
- **复述（Paraphrase）：** 用自己的话复述。「如果我没有听错，工厂关闭之后你大约有两年时间处于真正的不确定状态，是这样吗？」→ 叙述者确认或纠正。
- **情感反映（Reflect feeling）：** 说出你观察到的情绪基调。「听起来那段时间对你来说确实很令人恐惧。」不要说「我理解你的感受」——那会把焦点转移到采访员自身。
- **澄清（Clarify）：** 针对具体的模糊之处提问。「当你说「一切都变了」，什么是最先变的？」——打开空间，而非收窄空间。

下面的 **S.O.C.I.A.L.** 框架就是这三种回应方式在现场访谈中的操作化。

### C.3.B 跨文化与代际注意事项

| 情境 | 调整方式 |
|---|---|
| **老年受访者（70 岁以上）** | 放慢节奏；容许离题叙述（他们通常会绕回来）；不纠正日期；避免带有政治意涵的年代词汇 |
| **年轻采访员 / 老年受访者** | 主动摆出学习者姿态：「那个年代我还没出生——请帮我理解那是什么感觉。」避免将自己的参照系投射过去 |
| **方言使用者** | 如果受访者偏好用方言，鼓励他用；转写难度值得换取叙述真实性；方言舒适度往往与叙事深度正相关 |
| **受访者对采访员高度顺从（权力距离）** | 自我审查风险：坚定地重新定向——「我在这里是听你的记忆，没有所谓正确答案。」减少正式感，不要把记录行为戏剧化 |
| **受访者本身是权威人物** | 可能「表演」而非「回忆」；用具体、个人、非公开性的问题打破公众形象模式 |

### C.4 S.O.C.I.A.L. 追问框架

当受访者的回答值得进一步深入时，使用以下步骤：

| 步骤 | 动作 | 最佳时机 |
|---|---|---|
| **S**ilence 沉默 | 等待 5–10 秒，给对方空间 | 任何回答后；敏感话题后 |
| **O**bserve 观察 | 注意非语言线索：表情、坐姿、视线、手势 | 始终 |
| **C**larify 澄清 | 锁定模糊指称：「你说的「那时候」是指哪段时间？」 | 任何模糊词语出现时 |
| **I**nquire 追问 | 针对性探问：「你能多说说为什么做了那个决定吗？」 | 关键时刻刚浮现时 |
| **A**bstract-Concrete 抽象-具体切换 | 换层级：「整体上看，那段时间对你是怎样的？」/「给我一个具体的场景。」 | 话题引入后 |
| **L**ink 关联 | 连接线索：「这和你之前说的 X 有什么关系吗？」 | 出现主题性关联时 |

### C.5 动态调整大纲（在纸质大纲边缘手写）

| 信号 | 动作 |
|---|---|
| 受访者在某话题明显兴奋/滔滔不绝 | 追加深挖；把后续计划问题合并到这里 |
| 受访者在某话题明显回避 | 标 [暂缓]；不强求；在追访阶段再试 |
| 受访者提到完全没预料的信息 | 立刻在空白处写「追问该信息」 |
| 已有问题已不适用 | 划掉；在大纲末尾记「为什么没问」 |
| 疲劳迹象（60 分钟以上） | 进入收尾流程；保留 10 分钟给开放性兜底问题 |

**Talese 原则：** 「即使对方在努力表达时也绝不打断，因为正是在这些停顿、不精确的时刻……人是最真实的。他们犹豫不说的，往往最能说明他们是谁。他们的停顿、回避、突然跳转话题，通常是他们感到尴尬、恼怒，或认为过于私密的东西的指示。」（Gay Talese，《一个非虚构作家的起源》）

→ 沉默、迟疑、回避、话题跳转 = 数据，而非失败。

### C.6 特殊场景应对

| 场景 | 应对 |
|---|---|
| 受访者哭泣 | 暂停；递纸巾；等对方示意继续。不要说「没事的」「别哭」。改说：「需要休息一下吗？」 |
| 受访者愤怒 | 不评判、不打断；承认情绪：「听起来那段时间对你很有分量。」 |
| 受访者拒绝回答 | 「你不想说这个完全没问题。我们可以换话题，或者今天就到这里。」 |
| 受访者离题 | 让他说完；再说：「让我把你刚才说的核心点整理一下。」（离题内容往往是最有价值的——不要打断。） |
| 受访者说「不知道」 | 「不知道」本身就是信息：「你说你不太确定——这让你想到了什么？」 |
| 第三方进入（家属/同事） | 礼貌请对方在外等候；若坚持留下，记录于档（有第三方在场）；不将其发言纳入受访者证词 |
| 设备故障 | 立即停止录制；记录时间戳；向受访者说明；访谈结束后立即凭记忆写采访员笔记 |

### C.7 录制结束后立即执行（5 分钟内）

- [ ] 将录音备份到 2 个不同位置（本地 + 云盘）
- [ ] 写**采访员笔记**：你对这次访谈的情感反应、你自己的直觉和判断、你注意到的追访线索、任何未进入录音的内容
- [ ] 标记纸质大纲：实际问了哪些问题、顺序如何、跳过了哪些以及原因

**采访员笔记的意义：** 这与转写稿独立存档，代表采访员的主观视角——而非受访者的话语。它是档案的独立层。（OAH Best Practices：「项目的'代价'不只是财务上的……还有采访员、项目经理和团队成员的情感劳动。」）

## D. 追访

### D.1 追访触发条件表

| 触发情形 | 举例 | 评估时机 |
|---|---|---|
| 发现新线索 | 「我那个老师傅，后来去了 X……」（提到但未展开） | 访谈当天复盘时 |
| 发现矛盾 | 同一事件在不同场次讲述不同 | 转写整理时 |
| 需第三方核实 | 涉及某关键事件但当事人尚未被访谈 | 分析阶段 |
| 受访者主动补充 | 「我又想起一件事……」（访谈后发消息） | 立即回应；安排追访 |
| 受访者态度变化 | 初次回避某话题，之后主动联系 | 视情况判断 |
| 整理阶段发现缺漏 | 复听录音发现关键问题没问 | 转写整理后 |

**原则：** 「没有追访的项目是不完整的——90% 的好素材在追访中浮现。」

### D.2 追访技巧

- 比初访更有针对性——不需要大量的暖场
- 用「上次你说 X，我好奇 Y」开场——受访者会感觉被认真对待
- 避免「被审问」感——节奏应比初访轻；时长可以更短（30–45 分钟）
- 考虑带上初访笔记展示——受访者看到自己的话被认真对待会更开放
- 不重复已有好答案的问题；只聚焦未解决的线索

**关键区分：**
- **追问**（当次访谈内）：对刚才的回答深入追究——属于模式一执行技巧
- **追访**（另起一次访谈）：另约访谈以填补缺漏、解决矛盾、探索新线索——属于项目生命周期管理

### D.3 追访前核查清单

- [ ] 本次追访是否有明确的具体目标？（避免「再聊聊」式无目的追访）
- [ ] 受访者是否主动同意再次访谈？（非被动邀请）
- [ ] 时间间隔是否合理？（通常为初访后 2–4 周）
- [ ] 是否准备了新的简化大纲？（通常比初访更短）

**推荐音频格式：** 48 kHz / 24-bit WAV 或 FLAC。存档母带不使用 MP3 有损压缩。长期存储：音频用 WAV/FLAC；文字用 Markdown 或 PDF/A。

## H. 采访员成长

**需要纠正的误区：** 访谈不是「提问」。访谈是以高超的倾听技能，让叙述者感到足够安全，从而说出真实的话。

- **每次访谈后：** 5 分钟内写采访员笔记（§C.7）。30 分钟内第一印象就会消退——不要相信记忆。
- **每季度自我复盘：** 回听 2–3 次自己的访谈录音。问：我习惯性多问什么？我倾向回避哪些话题？我填充沉默的速度是否太快？
- **同行评审：** 与一位可信的同行分享一次访谈录音，邀请批评而非赞美。盲点的定义就是自己看不见。
- **研读自己的转写稿：** 标注自己打断、投射或引导的地方。没有采访员对这些习惯免疫。
- **情感自我关照：** 口述史访谈是情感劳动（OAH）。处理流离、悲伤或创伤叙事对采访员有累积性影响。承认这份重量；不要默默吸收它。
