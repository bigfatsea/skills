## 3. 判定规则（发现 → OK / WARN / FAIL）

| 发现 | 判定 | 理由 |
|---|---|---|
| <tool> 在 PATH 且唯一，无旧管理器 init 残留 | OK | 达标 |
| <tool> 与旧管理器 init 并存生效 | FAIL | shim/PATH 冲突，解析漂移 |
| 旧管理器目录残留但 init 已删 | WARN | 无害但建议清理，防止误 source 复活 |
| <tool> 缺失但语言仅一份、来源单一 | WARN | 可用但非推荐路线 |
| 同一 <tool> 在 PATH 出现多份 | FAIL | 版本可能停留在旧的那份而不自知 |
